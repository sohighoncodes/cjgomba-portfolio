# Logo asset library

Original logo files supplied by CJ Gomba, renamed for predictable website use.

- `tools/*-icon.*` — compact marks for badges and workflow nodes
- `tools/*-wordmark.*` — full horizontal or labeled logos
- `clients/penot-wordmark.png` — Penot Double Glazing client logo

These assets are intentionally stored as originals. Any web-optimized versions
should be generated separately so the supplied source files remain untouched.
