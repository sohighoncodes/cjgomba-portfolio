# CJ Gomba Portfolio

A responsive, static portfolio website for CJ Gomba.

The homepage provides short project summaries. Detailed case studies live in `projects/`
and use a consistent Problem, Planning, Goal, Solution, and Result structure.

## Preview

Open `index.html` in a browser or serve this folder with any static web server.

## Before publishing

1. Add CJ's email address to `CONTACT_EMAIL` at the top of `script.js`.
2. Add the final LinkedIn, Upwork, and booking links when available.
3. Replace the CJ monogram artwork with the final halftone headshot if desired.
4. Add the supplied testimonials, resume, and experience timeline once those assets are available.

The site does not include original client screenshots, so private customer details, live links,
workspace IDs, and internal records are not exposed.
